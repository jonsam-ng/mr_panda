# 字符符号收集 - Mr. Panda

---
在写作中加入字符符号可以增加写作的趣味性，这里收集了很多有趣的字符符号，做简单的分类。使用也很简单，直接复制字符图标到剪贴板，随处可粘贴。你可以把它当做图片来用，或者当做文字来用。图标的颜色不可改变，需要改变大小时直接用\`font-size\`即可。您可以在如下的场景中使用字符符号：

1.  git 工作流中。您可以在 git comment 中使用字符符号来乡镇您当前正在进行的工作。
2.  文章写作中。可以用作装饰，也可以用作记号，使用在 html 中比图片要节省流量。
3.  日常聊天中。这里就不用说了。。。
